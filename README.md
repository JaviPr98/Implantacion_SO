### Descripción
Este repositorio contiene la primera entrega práctica de la asignatura. En el documento PDF adjunto se documenta la resolución de los ejercicios sobre:
* Gestión de permisos, usuarios y grupos en Ubuntu Server (Modelo UGO).
* Administración avanzada de identidades y permisos especiales (SETGID, Sticky Bit).
* Creación de scripts de auditoría en Bash.
